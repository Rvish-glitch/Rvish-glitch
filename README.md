## Projects

**[Nutrition Scanner and Tracker](https://github.com/Rvish-glitch/nutripy)** — Real-time nutrition tracking with barcode scanning, meal history, and BMI computation. Built with FastAPI, async I/O, and in-memory SQLite indexing across 1000+ food records.

**[Plant Leaf Disease Predictor](https://github.com/Rvish-glitch/leaf-disease)** — Deep learning model (EfficientNetB3) achieving 99.59% accuracy across 38 disease classes and 15+ crop types. Flask REST API with sub-2s inference latency, deployed on Railway + Vercel.
